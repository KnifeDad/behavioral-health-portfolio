# 🌿 Behavioral Health Portfolio

> *Where clean code meets calm vibes.*  
> A portfolio for **Guy Ricketts** — websites for therapists & psychiatric providers who deserve something as good as their care.

---

## ✨ What's in the box?

This repo is a **single-page portfolio** built with React + TypeScript + Vite + Tailwind. It's built to show off work for independent clinicians: solo practices, group practices, and everyone in between.

- **Hero** — Who you are, what you do, and a very round photo.
- **Who I Serve** — Therapists, PMHNPs, small practices, telehealth folks.
- **Featured Projects** — FHCL Psychiatry + The Bloom Room Therapy (and room for more).
- **Services & Packages** — Solo Practice Launch vs Full Practice Website, with clear pricing vibes.
- **Process** — Consultation → Design → Build & Review → Launch (with numbered circles that match the brand).
- **Why Work With Me** — Short, trust-building copy.
- **Contact** — Form that actually sends stuff to your inbox (via Formspree).

Plus: dark theme, teal–orange gradients, and floating particles so it doesn't feel like a boring PDF. 🎈

## 🚀 Run it yourself

```bash
# Install dependencies
npm install

# Start the dev server (usually http://localhost:5173)
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview
```

Stick your own images in `public/` and your face will appear. Magic. 🪄
